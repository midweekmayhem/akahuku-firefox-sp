sp_rev.39 (2016-10-08)

  • WebM 対応
    • 「フォーム」の「添付ファイルをプレビュー」を動画にも対応 (ただし.webmがプレビューされるのはwebm対応板でのみ)
    • 「画像を保存」で動画の保存に対応
    • 「オートリンク」で動画も[見る][保存]可能に
    • 「MHTで保存」で動画のインライン再生要素を削除して整形するように
    • 「MHTで保存」で「元画像も保存する」有効時に動画も保存可能なように
    • 「レス一般」の「GIF画像を赤字で表示」有効時に動画も赤字で表示するように
    • インライン再生時の音量を「音」「ページ内メディアのデフォルト音量」で設定可能に(Firefox自体の設定"media.default_volume"を制御、Firefox 49以降のみ)
  • 新機能: カタログのソート時に特定のスレを一番上にピン止めする
    • 合間合間に等のような他拡張機能向けにフラグとなる属性を定義
    • オプション追加: 基本機能の使用例として、スレ本文を任意の文字列or正規表現で検索してフラグを立てる「本文で分ける」オプションを追加
  • 新機能: 設定「カタログ1」にレスの幅を固定するオプションを追加 (2chan.netのみ)
    • カタログの設定(ふたば側)で文字数を長くしてもレイアウトが崩れないようになり、また本文の表示行数を任意行数に制限できる(非表示部はスクロール可)
    • サムネもレスの幅と同じ縦横サイズの表示領域を確保して必要なら拡大するオプション
  • オプション追加: 「画像を保存1」で「保存した画像の表示([元画像]ボタン)の最大サイズ指定でビューポート相対値(vh,vw)を選択可能に
  • 修正・改善・調整
    • 修正: サーバー環境の変更以降?[MHTで保存]でCSSファイル等が保存できなくなった問題(Error in deflate (cache))
    • 修正: カタログでUndo/Redoすると既に開いているスレをクリックしてもタブ移動しない場合がある(まだ完全ではない?)
    • 修正: タブを即閉じるなどしていると希に機能停止に陥る問題 (TypeError: dead object)
    • カタログの更新のレスポンス改善(訪問履歴チェックを可能なら後回しにする)
    • カスタムCSSの登録について、Firefox 48以降でStylish等と同様のファイルを使わない手法に変更(開発ツールのInspectorがセキュリティエラーで正常動作しない問題への対処)
    • e10s対応の改善(画像を保存ボタンのメニュー位置, File I/O, 例外処理,...)
  • 板情報更新
